1A. function SetGameData (Max Life, Life Up, Max Soul, Soul Up, Exact Soul,)

Max Life [lifeMax]: How much life the character starts with.

Life Up: Life gained with each level.

Max Soul [soulMax]: How much soul the character starts with.

Soul Up: Soul gained with each level.

Exact Soul [soul = this.playerData.(Character).soulmax]: Defines which character gets which stats for their Max Soul.
