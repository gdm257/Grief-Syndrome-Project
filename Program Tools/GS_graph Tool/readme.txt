The GS_graph program, which originally was downloaded from here ( https://github.com/GriefSyndromeModderTools/GS_graph ), can be used for sprite extraction and importing. It also is useful for extracting the palette files.

It can primarly be used by first opening the command prompt program, then dragging and dropping the program onto the command prompt. Using the "GS_graph Commands.txt" as a guide. When it comes to the [path to file] and/or [path to folder] instructions, simply dragging and dropping the file or folder into the command prompt will work, without having to type out the entire path. Just make sure the commands, which are represented as (-a, -r, -h, -p, -m, -n, -f, -d) are placed where they are needed. The order is very important for this process, otherwise it won't work as intended.
