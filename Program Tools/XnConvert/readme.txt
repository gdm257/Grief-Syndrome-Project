The xnconvert program is capable of bulk image conversion.

The basic instructions boil down down to using the "Add files..." or "Add folder..." buttons on the Input tab.

Then, adjust the settings on the Output to get the desired outcome in terms of what you are looking for. For the sake of simplicity, just dajust the output file name to be what you want. The "{Filename}" option will automatically give the files whatever file name they originally had, with the exception of the format output. Any addition inputs that are put before or after that statement will just be added within the context of the file, but always before the format name.
