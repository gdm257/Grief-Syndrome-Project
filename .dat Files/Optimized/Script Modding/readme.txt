A compiled file containing almost all of the scripts from the "GSSquirrelScript" folder. The only current exception would be the "StagePause" folder and the "StagePause.global.cv4" file within, as they cause the game to crash if the game is paused while this .dat file is in use. The primary purpose for this .dat is to optimize modding progression by singling out specefic files while making acessability to the scripts themselves more manageable.
